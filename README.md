# Mintlify Landing Page Clone – Assignment

This project is a frontend clone of the Mintlify landing page(https://www.mintlify.com/).

The goal of this assignment was to recreate the layout design, typography, spacing, visual hierarchy, and UI struture using HTML and CSS.

---

🛠 Technologies used
- **HTML5** – page structure  
- **CSS3** – styling, layout, hover effects,flexbox
 
---

## Sections Recreated
1. Header / Navigation Bar
2. Hero Section
3. Partners Section
4. Features Section
5. Enterprise Section
6. Call-To-Action Section
7. Footer
